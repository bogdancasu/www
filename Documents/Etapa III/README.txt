Nume echipa: FireFly
Nume proiect: Odyssey


Descriere idee:
	
	Proiectul nostru isi propune sa imbunatateasca interactiunea intre utilizatori pe durata 
planificarii unei excursii, drumetii sau iesiri in oras, realizarea pasilor de planificare 
intr-un timp cat mai scurt, cat mai comod si simplu de folosit pentru oricine.

	Cu aceasta aplicatie, prietenii de pe facebook iti pot fi alaturi in planificarea unei 
excursii chiar si de la distanta, va puteti administra locatiile, listele de produse pe 
care le luati in excursie si puteti discuta topic-uri diferite, aceste elemente putand 
fi accesate si dupa relogare.

	
Functionalitati:
	1) Logare cu facebook:
		- preluare nume utilizator
		- preluare avatar utilizator
		- preluare lista de prieteni
	2) Trip Suggestions:
		- poze placute
		- scurta descriere a locatiilor
	3) New Trip:
		- cautare locatii pe harta (Google Maps)
		- adaugare data de plecare pentru fiecare locatie
		- adaugare prieteni de pe facebook si posibilitatea de a-i sterge
		
	4) My Trips:
		- doua zone disponibile utilizatorului
		- topics: zona de discutii cu prietenii 
		- topics: adaugarea de noi tab-uri de conversatie
		- assignments: adaugare de noi todo-uri pe sticky-notes
		- assignments: sticky-notes cu titlu, content si culoare editabile

Elemente folosite:
	-> Bootstrap library
	-> Google maps v3 API
	-> sticky-notes jQuery plugin
	-> Facebook API
	
Limbaje:
	-> html
	-> javascript
	-> jQuery
	-> html5
	-> css
